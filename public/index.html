Automatic backups

The site snapshots your data once a day automatically, kept for 30 days — this is a safety net for in-app mistakes, not a replacement for occasionally clicking Export data and saving that file somewhere off the site.

Restore from backup

If data ever gets wiped, upload an Export data file (or a downloaded backup from above) here to restore it. This replaces everything currently in the site — your current state gets snapshotted first as a safety net, but only use this when you mean to overwrite.
